# Annotation and Evaluation Artifacts

This folder contains the compact benchmark and evaluation artifacts submitted
with the thesis. It keeps the final joint benchmark, BRL evaluation outputs,
baseline outputs, ablations, confidence intervals, corpus-quality audit inputs,
and the small LLM passage-baseline experiment discussed in the thesis.

## Main Contents

- `joint_reference_gold_v1/`: the main 25-document joint benchmark and the
  evaluation artifacts used for the thesis tables.
- `llm_challenge_gold/`: routed law-reference hard-case benchmark artifacts.
- `llm_challenge_eval/`: supporting law-reference hard-case evaluation inputs.
- `llm_challenge_tasks/`: task files used to construct the routed hard-case
  benchmark.

Historical pilot folders, early experiments, and old prompt-fix runs are not
included in this compact submission package.

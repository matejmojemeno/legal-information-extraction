# Sample Corpus

This directory contains a tiny sample of normalized input texts in the same
source-family layout as the full thesis corpus:

- `parent_bucket_texts/nalus/`
- `parent_bucket_texts/ns/`
- `parent_bucket_texts/nss/`
- `parent_bucket_texts/uohs/`

The sample is included only to demonstrate the expected input shape. It is not
the evaluation corpus and it is not the full corpus-scale dataset used for the
final thesis run.
